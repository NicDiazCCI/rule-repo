# rule-repo

A TypeScript project with utility functions and testing infrastructure.

## Installation

```bash
npm install
```

## Scripts

- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode
- `npm run build` - Build TypeScript to JavaScript
- `npm run clean` - Remove build artifacts

## Project Structure

- `src/` - Source code
  - `utils.ts` - Utility functions including random boolean, delays, and flaky API calls
  - `__tests__/` - Test files

## Technologies

- TypeScript
- Jest (testing)
- CircleCI (CI/CD)
